# UAV Runtime MVP

轻量可测试的 UAV Runtime skeleton，包含：

- canonical 协议映射
- mission/scene 规划
- 适配器网关
- 2D 仿真与评估
- CLI 与简单 API facade

## 快速开始

```bash
python -m pip install -e .[dev]
pytest -q
python -m uav_runtime.console.cli health
```
