# 安全与合规说明（建议模板）

版本：v0.1（Demo）

> 说明：以下为公司化交付的“合规与安全”文档模板，不构成法律建议。

## 1. 使用边界

- 本平台定位于民用科研、巡检、搜救演训、游戏化仿真等用途。
- 禁止用于任何违法、危险或侵害人身财产安全的用途。

## 2. 风险与控制措施

### 2.1 LLM 安全

- LLM 输出必须为可审计的 JSON（decision/protocol），不直接控制执行器
- 对所有模型输出强制：schema 校验 + 规则约束检查 + 仿真回归
- 引入“安全策略层”（Safety Policy Layer）：地理围栏、禁飞区、速度/高度限制、权限

### 2.2 设备安全

- 适配器层必须实现：紧急停止、人工接管、指令白名单、速率限制
- 网络安全：TLS、设备证书、消息签名、重放保护（seq+timestamp）

### 2.3 数据合规

- 明确数据来源、脱敏策略、留存策略
- 对外提供日志/审计导出能力
